# 🔐 SpringBoot Login API

A secure and production-ready Login & Authentication REST API built using **Spring Boot** with **BCrypt password encryption**.

This project demonstrates how to implement a secure user authentication system using modern backend practices.

---

## 🚀 Features

✅ User Registration  
✅ Secure Login Authentication  
✅ Password Encryption using BCrypt  
✅ RESTful API Design  
✅ Maven Based Project  
✅ Clean & Scalable Structure  

---

## 🛠️ Tech Stack

- Java
- Spring Boot
- Spring Security
- BCrypt
- Maven
- REST API

## 🔑 Security

Passwords are securely stored using **BCrypt hashing**, ensuring:

- No plain text password storage
- Protection against brute force attacks
- Industry-standard encryption
